Background

Through conducting internal research, we learned that 80% of participants regularly create imagery for both internal and external use and are looking for a tool that can help them generate images quickly. Many users struggled to find visuals that align with Colgate's brand guidelines and often face challenges when editing images to meet specific needs. Due to this challenge we developed a series of guardrails to help users generate images that would align with Colgate's brands and AI image policies.

Original Design Studio Tool Competitive Analysis

The previous experience lacked essential functionality—users couldn't edit prompts or images, which forced them to start over each time they wanted to make changes. There was no library to access previously generated images, and no built-in safeguards to prevent users from creating content that violated company guidelines. It was far below the standard for AI image generators, and therefore we conducted a competitive analysis to help discover the best practices for designing image generators. Following that analysis we gathered as a team and labeled different aspects of the tools that we would like to implement into our own image generator.

Piece of the image gen design system

Approach

Once we had a clear structure in place, we moved on to wireframing, focusing on how each screen would support different user flows. This phase allowed us to quickly explore layout ideas without getting bogged down in visual design details. Early on, our main priority was determining how to organize tools between the prompting and editing stages, as well as defining the governance flow. This foundation enabled us to scale the product thoughtfully and introduce additional features over time. As part of the governance system, we implemented a check that activates when users upload a reference image for generation, helping ensure alignment with brand guidelines. After finalizing the user flow and wireframes we designed an entire design system around the new AI image generator, where we focused on modular design and reusability.

After the wireframes were finalized, we moved into high-fidelity UI design, translating our concepts into a visual system ready for developer handoff. This involved defining the web app's visual identity in alignment with Colgate's brand guidelines, while ensuring consistency with modern UI best practices. One of our key achievements was designing an experience that was both intuitive for beginners and robust enough for power users. We focused on creating a "learnable" platform—one that encouraged exploration and experimentation, helping users discover the right tools to generate their best possible images.
